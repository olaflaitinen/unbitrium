# Tutorial 147: Writing FL Research Papers

This tutorial covers best practices for documenting FL experiments.

## Sections

1. **Experimental Setup**: Datasets, models, baselines.
2. **Heterogeneity Description**: Partitioning, metrics.
3. **Hyperparameters**: All values, search ranges.
4. **Reproducibility**: Seeds, code availability.

## Checklist

- Report mean and std over multiple seeds.
- Include compute resources used.
- Provide configuration files.

1. Common pitfalls in FL reporting.
2. FAIR principles for FL research.
