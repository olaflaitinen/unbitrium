# Tutorial 187: Reddit Dataset

This tutorial covers Reddit as an FL benchmark.

## Task

Comment classification, sentiment, moderation.

## Features

- Subreddits as natural clients.
- Topic heterogeneity.

## Configuration

```yaml
dataset:
  name: "reddit"
  split_by: "subreddit"
```

## Exercises

1. Privacy considerations for social media.
2. Content moderation use cases.
