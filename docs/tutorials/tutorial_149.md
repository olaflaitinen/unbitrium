# Tutorial 149: Production FL Deployment

This tutorial covers deploying FL to production.

## Components

1. **Model Server**: Serves global model.
2. **Client SDK**: Embedded in apps.
3. **Coordinator**: Manages rounds, schedules.
4. **Monitoring**: Tracks health, metrics.

## Considerations

- A/B testing federated updates.
- Rollback strategies.
- Canary deployments.

## Exercises

1. CI/CD for FL models.
2. Model versioning strategies.
