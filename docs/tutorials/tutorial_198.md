# Tutorial 198: Federated Linear Models

This tutorial covers linear models for FL baselines.

## Models

- Logistic Regression.
- Linear Regression.
- SVM (linear kernel).

## Benefits

- Convex optimization.
- Strong convergence guarantees.

## Configuration

```yaml
model:
  type: "logistic_regression"
  regularization: 0.01
```
