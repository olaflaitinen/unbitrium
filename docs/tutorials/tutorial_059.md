# Tutorial 059: Gradient Leakage Attack (DLG)

## Overview
Deep Leakage from Gradients (Zhu et al., 2019). Reconstructing input image from gradient.

## Tutorial
Demonstrate the reconstruction loop on a single dummy gradient using `ub.privacy.attacks.DLG`.

## Mitigation
Show how minimal DP noise ($\epsilon=100$) or Gradient Clipping disrupts the reconstruction.
