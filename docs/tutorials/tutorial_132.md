# Tutorial 132: Federated Finance Applications

This tutorial covers FL in financial services.

## Use Cases

- Fraud detection across multiple banks.
- Credit scoring without sharing customer data.
- Anti-money laundering (AML) collaboration.

## Regulatory Context

- GDPR, PCI-DSS, SOX compliance.
- Data sovereignty requirements.

## Configuration

```yaml
domain: "finance"
encryption:
  enabled: true
  method: "secure_aggregation"
```

## Exercises

1. Feature engineering across heterogeneous banking systems.
2. Model interpretability requirements in finance.
