# Research References

This directory contains bibliographic information and summaries of key research papers implemented or referenced in Unbitrium.

## Structure

*   `bibliography.bib`: BibTeX file containing all citations.
*   `summaries/`: Markdown summaries of key papers (Work In Progress).

## Key Papers

1.  **FedAvg**: McMahan et al., "Communication-Efficient Learning of Deep Networks from Decentralized Data" (AISTATS 2017).
2.  **FedProx**: Li et al., "Federated Optimization in Heterogeneous Networks" (MLSys 2020).
3.  **FedDyn**: Acar et al., "Federated Learning on Non-IID Data via Dynamic Regularization" (ICLR 2021).
4.  **FedOpt (FedAdam)**: Reddi et al., "Adaptive Federated Optimization" (ICLR 2021).
5.  **Secure Aggregation**: Bonawitz et al., "Practical Secure Aggregation for Privacy-Preserving Machine Learning" (CCS 2017).

See `bibliography.bib` for full details.
