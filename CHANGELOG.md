# Changelog

All notable changes to Unbitrium will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-01-03

### Added
- **Production Release**: Unbitrium 1.0.0 is now stable and production-ready.
- Complete documentation suite with 200 interactive tutorials.
- Full API reference for all modules.
- Verified compatibility with Python 3.14.1.
- Final research notes corpus in `docs/research/notes.md`.
- Bibliography in BibTeX, CSL-JSON, and Markdown formats.

### Changed
- Standardized all docstrings to Google style with raw strings for LaTeX.
- Updated official documentation color to `#66023C`.
- Enforced standard header with Author: Olaf Yunus Laitinen Imanov.

### Fixed
- Resolved minor import issues in `afl_dcs` module (renamed class to `AFL_DCS`).
- Fixed escape sequence warnings in docstrings across the codebase.
