# Support

## Getting Help

If you have questions about using Unbitrium, need troubleshooting assistance, or want to discuss federated learning simulation topics:

1.  **Documentation**: Check the official documentation and the 200+ tutorials in `docs/tutorials/`.
2.  **GitHub Issues**: If you suspect a bug or have a feature request, open an issue on the [GitHub repository](https://github.com/unbitrium/unbitrium/issues).
3.  **Contact**: For research collaborations or specific inquiries, contact the project lead:
    *   **Olaf Yunus Laitinen Imanov**
    *   Email: `oyli@dtu.dk`
    *   Technical University of Denmark (DTU)
